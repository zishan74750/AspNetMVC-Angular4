# AspNetMVC-Angular4
A simple starter template for ASP.NET MVC with Angular 4.
If you want to use ASP.NET MVC with Angular 4, you can start with this template.

# Steps
 
-Clone project <br />
-Open it <br />
-Click right to package.json file and Restore Packages <br />
-Run!
